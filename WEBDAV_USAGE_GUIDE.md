## 🎯 使用指南

### 1. 配置备份目标

#### 本地存储
1. 进入"备份目标"页面
2. 点击"添加目标"
3. 选择类型：本地存储
4. 配置本地路径：`/app/backups`

#### WebDAV
1. 进入"备份目标"页面
2. 点击"添加目标"
3. 选择类型：WebDAV
4. 配置信息：
   - WebDAV URL: `https://cloud.example.com/remote.php/dav/files/username/`
   - 用户名
   - 密码
   - 远程路径: `backups/`

#### 目标服务器
1. 先在"源服务器"页面添加目标服务器
2. 进入"备份目标"页面
3. 点击"添加目标"
4. 选择类型：目标服务器
5. 选择已配置的服务器

### 2. 创建备份任务

1. 进入"备份任务"页面
2. 点击"创建任务"
3. 配置任务：
   - 任务名称
   - 源服务器
   - Cron 表达式
   - **选择多个备份目标**（可多选）
4. 保存任务

### 3. 执行流程

```
定时触发
  ↓
从源服务器导出数据
  ↓
同时备份到所有选中的目标
  ├─ 本地存储（复制文件）
  ├─ WebDAV（上传文件）
  └─ 目标服务器（导入数据）
```
