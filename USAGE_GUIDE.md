## 使用指南

### 1. 配置服务器

1. 访问 Web 界面
2. 点击"服务器配置"标签
3. 点击"添加服务器"按钮
4. 填写服务器信息：
   - **名称**: 服务器标识名称
   - **服务器 URL**: Bitwarden 服务器地址
   - **Client ID**: API 客户端 ID
   - **Client Secret**: API 客户端密钥
   - **Master Password**: 主密码
   - **官方服务器**: 勾选表示这是官方服务器

### 2. 创建备份任务

1. 点击"备份任务"标签
2. 点击"创建任务"按钮
3. 配置任务参数：
   - **任务名称**: 任务标识名称
   - **源服务器**: 选择要备份的服务器
   - **Cron 表达式**: 定时规则（见下方说明）
   - **备份路径**: 备份文件存储路径
   - **启用自动迁移**: 是否自动导入到目标服务器
   - **目标服务器**: 迁移的目标服务器（启用迁移时）
   - **启用任务**: 是否启用此任务

### 3. Cron 表达式说明

格式：`秒 分 时 日 月 周 年`

常用示例：
- `0 0 2 * * *` - 每天凌晨 2 点
- `0 0 */6 * * *` - 每 6 小时
- `0 30 1 * * 1` - 每周一凌晨 1:30
- `0 0 0 1 * *` - 每月 1 号凌晨

### 4. 查看执行日志

1. 点击"执行日志"标签
2. 查看备份任务的执行历史
3. 状态说明：
   - **success**: 执行成功
   - **failed**: 执行失败
   - **running**: 正在执行
