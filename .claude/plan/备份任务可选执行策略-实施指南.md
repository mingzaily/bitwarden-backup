## 6. 测试清单

### 手动测试

4. 编辑任务：添加 Cron 表达式 → 验证变为定时任务
5. 手动触发任务 → 验证执行成功
6. 重启服务 → 验证只加载定时任务

- POST `/api/tasks` 不传 `cron_expression`
- POST `/api/tasks` 传入无效 Cron 表达式
- PUT `/api/tasks/:id` 清空 `cron_expression`
- PUT `/api/tasks/:id` 添加 `cron_expression`
