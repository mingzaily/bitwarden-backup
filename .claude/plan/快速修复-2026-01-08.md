### 步骤 4：改进 Bitwarden unlock

#### 4.1 修改 Unlock 函数使用 stdin

**文件**: `internal/bitwarden/export.go`

**修改位置**: 第 11-27 行

**修改前**:
```go
func (c *Client) Unlock(masterPassword string) error {
	cmd := exec.Command("bw", "unlock", masterPassword, "--raw")
	output, err := cmd.CombinedOutput()
	if err != nil {
		return fmt.Errorf("unlock failed: %s, %w", string(output), err)
	}

	c.sessionToken = strings.TrimSpace(string(output))

	if c.sessionToken == "" {
		return fmt.Errorf("unlock returned empty session token, output: %s", string(output))
	}

	return nil
}
```

**修改后**:
```go
func (c *Client) Unlock(masterPassword string) error {
	cmd := exec.Command("bw", "unlock", "--raw")

	// 使用 stdin 传递密码，避免暴露在命令行参数中
	cmd.Stdin = strings.NewReader(masterPassword)

	output, err := cmd.CombinedOutput()
	if err != nil {
		return fmt.Errorf("unlock failed: %s, %w", string(output), err)
	}

	c.sessionToken = strings.TrimSpace(string(output))

	if c.sessionToken == "" {
		return fmt.Errorf("unlock returned empty session token, output: %s", string(output))
	}

	return nil
}
```

#### 4.2 添加 bw status 预检

**文件**: `internal/bitwarden/export.go`

**新增函数**（在 Unlock 函数之前）:
```go
// Status 检查 Bitwarden CLI 状态
func (c *Client) Status() (string, error) {
	cmd := exec.Command("bw", "status")
	output, err := cmd.CombinedOutput()
	if err != nil {
		return "", fmt.Errorf("status check failed: %w", err)
	}
	return string(output), nil
}
```
